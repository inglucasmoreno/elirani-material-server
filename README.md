
## Instalación

```bash
$ yarn install
```

## Correr la aplicación

- Configurar las variables de entorno primero que todo

```bash
# Desarrollo
$ yarn start

# Modo seguimiento
$ yarn start:dev

# Modo produccion
$ yarn start:prod
```
